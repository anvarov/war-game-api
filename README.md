# war-game-rest

War game rest api

## Dependencies

Docker

## How to use
`git clone https://github.com/anvarov/war-game-api.git`  
`cd war-game-api`  
`docker build .`  
`docker run -p 3000:3000 -d <image_id>`  

# API

APIs available at `localhost:3000`
Endpoints:
 - `/simulateGame` - simulates a game and returns a winner
 - `/getPlayers` - get players stats
